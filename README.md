# 🌍 All Countries Explorer

A fully responsive country explorer built with **Next.js (App Router)** and **Tailwind CSS**.  
It allows users to browse, search, filter, and view detailed information about countries with an interactive map.

---

## 🔗 Live Demo

👉 https://all-countries-next.vercel.app

---

## ✨ Features

- 🌍 View all countries in a clean UI  
- 🔍 Search countries by name  
- 🌎 Filter by region  
- 📄 Detailed country information page  
- 🗺️ Interactive map integration  
- 🌙 Dark/Light mode support  
- ⚡ Fully responsive (mobile + tablet + desktop)

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- REST Countries API
- React Leaflet (Map Integration)
